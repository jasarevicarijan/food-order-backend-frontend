# Food Order App With JS & React

This is a repository for a Food Order App using JS and React.

Features:

- Backend
- Frontend
  
### Cloning the repository

```shell
git clone https://github.com/jasarevicarijan/food-order-backend-frontend.git
```

### Install packages

```shell
npm install
```
```shell
cd backend
```
```shell
npm install
```

### Start the app

```shell
cd backend
```
```shell
npm start
```
New Terminal
```shell
npm run dev
```
